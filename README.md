# Python_BasicProjects

Autor: Jonathan Osiris González Ibarra  
Lenguaje: Python  
Nivel: Básico  
Repositorio de proyectos básicos desarrollados en Python con fines educativos, de práctica personal y fortalecimiento de la lógica de programación.

---

## Descripción General

Este repositorio contiene una colección de **10 proyectos básicos en Python**, diseñados para practicar los fundamentos esenciales del lenguaje, tales como entrada y salida de datos, uso de variables, condicionales, ciclos, funciones, listas, librerías estándar y lógica de programación.

Cada proyecto es una aplicación de **consola interactiva**, ideal para estudiantes que están iniciando en Python.

---

## Proyectos Incluidos

### 1. Calculadora Básica  
Permite realizar operaciones aritméticas básicas:
- Suma
- Resta
- Multiplicación
- División  
Incluye interacción con el usuario y simulación de carga.

---

### 2. Adivina el Número  
Juego interactivo donde:
- El sistema genera un número aleatorio.
- El usuario intenta adivinarlo.
- Se muestra si acertó o falló.  
Se utiliza la librería `random`.

---

### 3. Conversor de Unidades  
Permite convertir:
- Metros a Kilómetros  
- Grados Celsius a Fahrenheit  
- Pesos a Dólares  
Incluye un menú de selección para el usuario.

---

### 4. Registro de Usuarios  
Sistema que permite:
- Pedir nombre, edad y correo.
- Guardar los datos en una lista.
- Mostrar todos los usuarios registrados.

---

### 5. Contador de Vocales  
Solicita una palabra o texto al usuario y:
- Cuenta cuántas vocales contiene.
- Muestra el total de vocales encontradas.

---

### 6. Generador de Contraseñas  
Genera contraseñas aleatorias:
- Usando letras y números.
- Con longitud definida por el usuario.
- Utiliza la librería `random`.

---

### 7. Lista de Tareas (To-Do List)  
Permite al usuario:
- Agregar tareas.
- Mostrar las tareas guardadas.
- Eliminar tareas existentes.
Funciona mediante una lista en memoria.

---

### 8. Tablas de Multiplicar  
Muestra en pantalla:
- La tabla de multiplicar de un número ingresado por el usuario.
- Desde el 1 hasta el 10.

---

### 9. Piedra, Papel o Tijera  
Juego clásico donde:
- El usuario selecciona una opción.
- La computadora elige aleatoriamente.
- Se determina el ganador.

---

### 10. Sistema de Calificaciones  
Permite:
- Ingresar varias calificaciones.
- Calcular el promedio.
- Determinar si el alumno aprueba o reprueba.

---

## Tecnologías Utilizadas

- Python 3
- Librerías estándar:
  - `time`
  - `os`
  - `random`

No se utilizan librerías externas.

---
